## ⚙️ How It Works

1. A scheduler triggers twice daily (09:00 and 20:00).
2. An LLM generates comprehensive notes for the next AWS topic.
3. Notes, interview questions, CLI commands, and labs are created.
4. Everything is committed and pushed via the GitHub API.
5. This README is automatically updated with current progress.
